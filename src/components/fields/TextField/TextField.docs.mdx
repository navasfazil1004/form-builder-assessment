# TextField

`TextField` is a versatile form input component with:

- Validation rules (`required`, `maxLength`, `minLength`, `pattern`)  
- Character counter for maxLength  
- Clear button  
- Input masking support  

## Usage

```vue
<TextField
  v-model="firstName"
  :field="{ id: 'firstName', label: 'First Name', validation: [{ type: 'required', message: 'Required' }] }"
  placeholder="Enter your first name"
/>
